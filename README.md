# IoT-Based Accident Detection and Alert System

## Project Overview
This project is an IoT-based real-time accident detection and alert system using ESP32. It detects accidents using sensors and automatically sends the GPS location to emergency contacts through GSM and Blynk notifications.

## Features
- Real-time accident detection
- GPS location tracking
- Automatic SMS alerts
- Email notifications
- Blynk IoT monitoring
- Emergency alert system

## Hardware Components
- ESP32
- MPU6050 Sensor
- GPS Module
- GSM Module
- Buzzer
- Push Button
- Power Supply

## Software Used
- Arduino IDE
- Embedded C/C++
- Blynk IoT
- ESP32 Libraries
